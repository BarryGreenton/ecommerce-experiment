# 电子商务网站实验项目（网络应用架构实验）
葛林嵩 202330840441
> 本项目用于完成“设计并实现一个电子商务网站的开发和在线部署”的课程实验。

## 项目说明

- **技术栈**：Node.js + Express + SQLite + EJS
- **基本功能**：
  - 顾客：注册、登录、注销；商品列表浏览；购物车；下单/付款；查看订单历史
  - 销售（管理员）：商品目录管理；订单管理与销售统计；客户信息查看；浏览/购买日志记录


## 快速启动（本地）

```bash
cd ecommerce-site
npm install
npm start
```

启动后访问：`http://localhost:3000`

- 默认管理员账号：`admin`
- 默认管理员密码：`123456`
